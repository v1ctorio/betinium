# Betinium
2-person game where you need to move your phone to stop the explosion

Made with Godot, Expo and sleep deprivation

## Run

1. Start Expo

```
npx expo start
```

A QR Code will appear, make sure both devices are in the same network and scan it

Alternatively you can manually connect by typing the url
